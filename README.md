# ISPP-G1-Talent
## Quick start
Clone this repository
```bash
git clone https://github.com/JaviFdez7/ISPP-G1-Talent.git
```
Install the dependencies (from the root directory)
```bash
npm install
```
Create .env file from .env.example file
```
GH_TOKEN=<YOUR-GH-TOKEN>
```
Run the server
```bash
npm start
```
